## [Examples of calling the data](https://www.youtube.com/watch?v=1zzCD2WmYoE&t=484s)

The Unicorn has a base speed of  `= this.movement.unicorn.base` and takes `= this.movement.unicorn.normal` minutes to go 1 mile.

The Battle Balloon has a base speed of `=this.movement["battle balloon"].base` and takes `=this.movement["battle balloon"].normal` minutes to go 1 mile.

The Rhino has a base speed of `= this.movement[this.transportation].base` and takes `= this.movement[this.transportation].normal` minutes to go 1 mile.

### Table
```dataviewjs 
let pg = dv.current();
let table = "| Name           |base | slow | normal | fast |\n" +
            "| ------------  | ----------:| ---:|-----: | ---:|\n"; 
dv.header(3, pg.title);
for (let move of Object.entries(pg.movement)) {
    table += "| " + move[1].name +
            " | " + move[1].base +
            " | " + move[1].slow +
            " | " + move[1].normal +	
            " | " + move[1].fast +
            " |\n" 

}
dv.paragraph(table);
```
